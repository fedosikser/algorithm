Портфолио из 7 проектов на С++.

Ключевые компетенции
* Алгоритмы и структуры данных
* Анализ сложности (Big-O)
* Динамическое программирование
* Жадные алгоритмы
* Графы (BFS)
* Основы машинного обучения (kNN)
* Теория вероятностей и комбинаторика
* Проектирование логики приложений
* Работа с файлами и пользовательским вводом



Структура репозитория
Каждый проект размещён в отдельной папке. Название папки начинается с #, например: #dynamic_prog.
Внутри каждой папки находится файл README.md с подробным описанием:
* постановки задачи
* используемого алгоритма
* архитектуры решения
* анализа сложности
* примера работы программы
Ниже — краткая выжимка по каждому проекту.



Алгоритмы и структуры данных
1. Dynamic Programming — задача о рюкзаке (#dynamic_prog)
Реализация классической задачи оптимизации с использованием динамического программирования. Цель — найти максимальную суммарную ценность при ограничении по вместимости.
Что реализовано:
* Табличная модель DP
* Восстановление оптимального набора предметов
* Работа с входными данными из файлов
* Анализ временной и пространственной сложности
В папке проекта содержится README.md с подробным описанием алгоритма и примерами работы.

2. Greedy Algorithm — задача о покрытии множества (#approximate_greedly_algorithm)
Реализация приближённого жадного алгоритма для NP-сложной задачи (Set Cover).
Что реализовано:
* Выбор локально оптимального решения на каждом шаге
* Снижение сложности с O(2ⁿ) до O(n²)
* Работа с множествами и их пересечениями
* Анализ компромисса между точностью и скоростью
В папке проекта содержится README.md с подробным разбором задачи.

3. Поиск кратчайшего пути (BFS) (#find_short_way)
Алгоритм поиска в ширину для нахождения кратчайшего пути в графе.
Что реализовано:
* Очередь для обхода графа
* Поиск минимального расстояния
* Работа с входными данными из файла
В папке проекта содержится README.md с описанием реализации.

Машинное обучение
4. Система рекомендаций фильмов (k-Nearest Neighbors) (#k_closest_neighbor)
Реализация рекомендательной системы на основе алгоритма k ближайших соседей с использованием реальных данных IMDb.
Что реализовано:
* Определение предпочтительных жанров пользователя
* Прогнозирование оценки фильма
* Учёт индивидуальной строгости пользователя
* Подготовка к расширению модели (genome tags)
В папке проекта содержится README.md с описанием модели и логики работы.

Структуры данных и проектирование систем
5. Упрощённая модель Facebook (#list_facebook)
Реализация гибридной структуры данных для хранения и управления пользователями.
Что реализовано:
* Баланс между скоростью поиска и вставки
* CRUD-операции (Find / Insert / Erase)
* Проектирование структуры с учётом производительности
В папке проекта содержится README.md с описанием структуры данных.

Математика + симуляции
6. Card Games — модель казино (#card_games)
Проект, объединяющий структуры данных, теорию вероятностей и комбинаторику.
Что реализовано:

В папке проекта содержится README.md с описанием архитектуры и логики игр.

#approximate greedly algorithm 
Задача о покрытии множества. Жадный алгоритм. Выбираем наиболее оптимальный вариант каждый раз (близкий к оптиуму) без перебора всех комбинаций. Алгротмическая сложность O(n^2)  (вместо O(2^n) для алгоритма перебора всех комбинаций.)


Известен список станций и список штатов, которые могут быть покрыты каждой станцией. Необходимо найти минимальное количество станций, которые можно использовать для покрытия всех штатов. (или определенных штатов).

k_closest_neighbor
Система рекомендаций фильмов (как на Кинопоиск).
Реализация алгоритма k ближайших соседей. Машинное обучение на реальных данных от IMDb. 

Алгоритм позволяет опрделить любимый жанр фильма у пользователя, спрогнозировать оценку фильма у пользователя.
Алгоритм учитывает строгость пользователя к оценкам фильмов. Планируется внедрить учет genome tags (тегов фильма) для улучшения результатов алгоритма.

card_games
Карточные игры. Модель казино.

Реализованы структуры для хранения данных о пользователях и картах.

Реализована логика игры: Блекджек. На основе теории вероятности написал алгоритм для логики Диллера.

Реализована логика для игры: Слоты. На основе знаний из комбинаторики написал коэффецент выигрыша. 

Реализована система денежного баланса пользователей.

## 📂 Обзор проектов

### 1. Динамическое программирование — задача о рюкзаке (`#dynamic_prog`)
Классическая задача оптимизации: выбор предметов с максимальной ценностью при ограничении веса.  
**Реализовано:** табличный DP, восстановление набора, ввод из файла, анализ `O(n*W)`.

### 2. Жадный алгоритм — покрытие множества (`#approximate_greedly_algorithm`)
Приближённое решение NP-сложной задачи Set Cover.  
**Реализовано:** локально-оптимальный выбор за `O(n²)`, работа с пересечениями множеств.

### 3. BFS — кратчайший путь в графе (`#find_short_way`)
Поиск минимального расстояния в невзвешенном графе.  
**Реализовано:** очередь, обход в ширину, чтение графов из файлов.

### 4. Рекомендации фильмов — kNN (`#k_closest_neighbor`)
Система на основе реальных данных IMDb.  
**Реализовано:** определение жанровых предпочтений, прогноз оценки, учёт строгости пользователя.

### 5. Упрощённая модель Facebook (`#list_facebook`)
Гибридная структура данных для хранения пользователей.  
**Реализовано:** баланс поиска/вставки, CRUD (`Find`/`Insert`/`Erase`).

### 6. Карточные игры — модель казино (`#card_games`)
Объединение структур данных, вероятностей и комбинаторики.  
**Реализовано:** BlackJack (логика дилера), слоты (расчёт коэффициентов), баланс пользователя, авторизация.

### 7. Hi-Lo Game (`#hi_lo`)
Консольная игра «Угадай число».  
**Реализовано:** игровая логика, лимит попыток, управление состоянием.
